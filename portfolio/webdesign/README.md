# Web Design Portfolio

This section contains website templates, mockups, and demo sites for various business types, including painting and remodeling companies.

## Structure
- /templates: Ready-to-use HTML/CSS/JS templates
- /demos: Fully functional demo sites
- /assets: Images, icons, and design resources
- /docs: Design notes, style guides, and instructions

---

Start by adding your first template or demo site in the appropriate subfolder. For a painting and remodeling company, consider including:
- Home/Landing page
- Services page
- Gallery/Portfolio
- About/Team
- Contact/Quote form

If you need a starter template or a full demo, let me know your preferred stack or style!
